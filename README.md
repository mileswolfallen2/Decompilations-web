# Decompilations-web

An animated HTML, CSS, and JavaScript showcase for native PC ports, decompilations, and recompilations.

## What it includes

- A polished single-page interface with animated section switching.
- Dedicated views for fully released projects, active development work, and complete decompilations.
- A sources section with official repositories and legal reminder text.
- Detail cards with progress indicators and project links.

## How to help 

1. just share the project with others
2. help with dev work or bugs
3. help as keep it up to date with info
4. help add info 

## Notes

- The site uses Anime.js for page and card animations.
- Progress bars only appear when a project has a known progress value.
- No game assets are included. You must legally own and dump your own ROM or ISO files.
